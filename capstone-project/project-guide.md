# Capstone Project Guide

## Overview
The capstone project is the culminating experience of the Modern Systems Engineering curriculum. Students design and document a complete system architecture that integrates concepts from all seven courses.

## Choose One System to Build

1. AI Knowledge Platform
2. Uber Dispatch System
3. Global Chat Platform
4. Payment Processing Platform
5. AI Coding Assistant

## Deliverables

- Architecture diagram
- API design
- Scaling strategy
- Monitoring plan
- Working prototype

## Project Requirements
1. **Problem Statement**: Define a real-world engineering problem to solve.
2. **Architecture Design**: Produce a detailed system architecture with component diagrams.
3. **Technology Choices**: Justify the selection of technologies and trade-offs made.
4. **Scalability & Reliability**: Describe how the system scales and handles failures.
5. **Security**: Address authentication, authorization, and data protection.
6. **Observability**: Define monitoring, logging, and alerting strategies.
7. **Deployment**: Outline an infrastructure-as-code deployment plan.
