# Modern Systems Engineering Curriculum

A practical curriculum for learning **production-grade systems engineering** using real engineering case studies and hands-on labs.

This repository compiles engineering lessons from companies like:

* Netflix
* Uber
* AWS
* Cloudflare
* Stripe
* GitHub
* Discord
* OpenAI
* NVIDIA
* Shopify
* Figma

The goal is to help engineers learn how **large-scale systems actually work in production.**

---

# Curriculum Overview

| Course | Topic                    | Duration  |
| ------ | ------------------------ | --------- |
| 1      | Engineering Foundations  | 4 weeks   |
| 2      | Distributed Systems      | 5 weeks   |
| 3      | Cloud Infrastructure     | 4 weeks   |
| 4      | AI Systems Engineering   | 4 weeks   |
| 5      | Real-Time Systems        | 3 weeks   |
| 6      | Financial Infrastructure | 3 weeks   |
| 7      | Advanced System Design   | 4 weeks   |
| Final  | Capstone Project         | 4–6 weeks |
