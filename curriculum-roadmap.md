# Curriculum Roadmap

This roadmap outlines the learning path for the Modern Systems Engineering curriculum.

## Course Sequence

1. **Course 01 – Engineering Foundations**: Core principles of software and systems engineering.
2. **Course 02 – Distributed Systems**: Concepts, patterns, and real-world distributed architectures.
3. **Course 03 – Cloud Infrastructure**: Cloud platforms, infrastructure-as-code, and operations.
4. **Course 04 – AI Systems**: Machine learning systems, pipelines, and MLOps.
5. **Course 05 – Realtime Systems**: Event-driven architectures, streaming, and low-latency design.
6. **Course 06 – Financial Systems**: High-reliability and compliance-driven system design.
7. **Course 07 – Advanced System Design**: Large-scale system design patterns and trade-offs.
8. **Capstone Project**: End-to-end system design project integrating all course concepts.
