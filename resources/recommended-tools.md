# Recommended Tools

A curated list of tools and platforms used throughout the Modern Systems Engineering curriculum.

## Development & Collaboration
- [Git](https://git-scm.com/) – Version control
- [GitHub](https://github.com/) – Code hosting and collaboration
- [VS Code](https://code.visualstudio.com/) – Code editor

## Infrastructure & Cloud
- [Terraform](https://www.terraform.io/) – Infrastructure as Code
- [Docker](https://www.docker.com/) – Containerization
- [Kubernetes](https://kubernetes.io/) – Container orchestration
- [AWS](https://aws.amazon.com/) / [GCP](https://cloud.google.com/) / [Azure](https://azure.microsoft.com/) – Cloud providers

## Observability
- [Prometheus](https://prometheus.io/) – Metrics collection
- [Grafana](https://grafana.com/) – Metrics visualization
- [Jaeger](https://www.jaegertracing.io/) – Distributed tracing
- [ELK Stack](https://www.elastic.co/elastic-stack) – Log aggregation

## Messaging & Streaming
- [Apache Kafka](https://kafka.apache.org/) – Distributed event streaming
- [RabbitMQ](https://www.rabbitmq.com/) – Message broker

## Databases
- [PostgreSQL](https://www.postgresql.org/) – Relational database
- [Redis](https://redis.io/) – In-memory data store
- [Cassandra](https://cassandra.apache.org/) – Distributed NoSQL database

## AI / ML
- [Python](https://www.python.org/) – Primary language for ML
- [PyTorch](https://pytorch.org/) / [TensorFlow](https://www.tensorflow.org/) – Deep learning frameworks
- [MLflow](https://mlflow.org/) – ML lifecycle management
